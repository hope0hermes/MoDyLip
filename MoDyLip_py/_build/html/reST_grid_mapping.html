2D Grid Mapping¶

Discrete mapping of a scalar quantity over a 2D surface.

This class can be though as an weighted and non-normalized histogram. It is handful for a quick, coarse-grained classification and visualization of data.

class py3_grid_mapping.GridMap(domain, target, points=[32, 32])[source]¶
